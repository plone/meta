# Tutorials

<!-- diataxis: tutorials -->

Step-by-step lessons that guide you through using `plone.meta`.
Start with the basics and work your way up to managing multiple repositories.

::::{grid} 2
:gutter: 3

:::{grid-item-card} Configure Your First Package
:link: first-package
:link-type: doc

Clone plone.meta, run `config-package` on a repository, and understand
the files it generates. Learn how `.meta.toml` controls the output.

*Estimated time: 15 minutes*
:::

:::{grid-item-card} Update Multiple Repositories
:link: bulk-updates
:link-type: doc

Use `multi-call` to apply configuration changes across many Plone
packages at once. Learn the packages.txt format and the clone workflow.

*Estimated time: 10 minutes*
:::
